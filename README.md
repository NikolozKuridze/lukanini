# Nini x Luka Love Arcade

კერძო, 2-მოთამაშიანი, რეალურ დროში მუშაობადი Next.js + Tailwind საიტი.

## გაშვება

```bash
npm install
cp .env.example .env.local
npm run dev
```

## HTTPS (ლოკალზე მიკროფონისთვის რეკომენდებული)

```bash
npm run dev:https
```

## რას აკეთებს

- პაროლით დაცული შესვლა
- მაქსიმუმ 2 მოთამაშე (მესამე იბლოკება)
- IP რეკომენდაცია:
  - `FR` -> ნინი
  - `GE` -> ლუკა
- 3 თამაში თანმიმდევრობით:
  - ლოგიკური ქვიზი
  - Fire & Water კოოპ დაბრკოლებები
  - 3D სტილის ნარდი (visual)
- საუბრის ღილაკი (WebRTC აუდიო)
- მუსიკა ფონში + mute/pause კონტროლი
- მოულოდნელი პატარა ფოტო-პარტიკლები

## API

- `GET /api/geo`
- `GET/POST /api/room`
- `GET /api/room/events`
- `POST/DELETE /api/auth`

## შენიშვნა

- ბრაუზერის უსაფრთხოების გამო მიკროფონის უფლების **ავტომატურად დადასტურება შეუძლებელია**.
- მუსიკის სრული autoplay ხმიან რეჟიმში ზოგ ბრაუზერში იზღუდება, ამიტომ UI კონტროლი დამატებულია.
